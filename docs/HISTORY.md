# History

## OpenBrain v0.1-v0.3

- OpenBrain established the original local-first assistant architecture with n8n, Postgres, embeddings, retrieval, and a webhook-driven assistant flow.
- Those releases remain preserved in `/Users/elric/repos/openbrain`.
- The original OpenBrain concept introduced by Nate Jones inspired the early system direction.

## CrispyBrain v0.4

- CrispyBrain v0.4 forks from the OpenBrain lineage into a new repository at `/Users/elric/repos/crispybrain`.
- The fork keeps the underlying database schema compatible for now while rebranding the working system, simplifying imported workflow names, and shortening the assistant webhook from `openbrain-assistant` to `assistant`.
- OpenBrain and CrispyBrain are intended to coexist side-by-side in the same n8n instance.

## Acknowledgement

This fork preserves the original OpenBrain lineage while evolving it into a separately branded project. Nate Jones and the OpenBrain concept are acknowledged as part of that origin story.
